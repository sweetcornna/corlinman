the platform  /  平台

Everything an agent needs to stay.

记忆 durable memory

Hybrid RAG — vector, keyword, and a self-written diary. Context that survives every restart.

工具 real tools

A sandboxed plugin runtime with skills, schedulers, browsers, and document generation.

渠道 seven channels

QQ, Telegram, Discord, Slack, Feishu, WeChat — and a first-class web chat, all one agent.

审批 human-in-the-loop

Irreversible actions queue for your approval before they run. Govern, don't hope.

后台 operations plane

A Spatial Glass admin console — live logs, RAG debugging, node graphs, theme studio.

单体 one binary
